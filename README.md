# Web Development Project 5 - Data Dashboard

Submitted by: HJ Kim

This web app is a dashboard that provides users with information on brewries in the United States. Users can search brewries by the name or the location (state) of the brewery.

Time spent: 3 hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **The list displays a list of data fetched using an API call**
- [x] **Data uses the useEffect React hook and async/await syntax**
- [x] **The app dashboard includes at least three summary statistics about the data such as**

  - [x] total number of items in the dataset
  - [x] number of items that meet certain criteria (closed brewries, micro breweries )
  - [x] most frequent type of brewries among the search result

- [x] **A search bar allows the user to search for an item in the fetched data**
- [x] **Multiple different filters (2+) allow the user to filter items in the database by specified categories**

The following **optional** features are implemented:

- [x] Multiple filters can be applied simultaneously
